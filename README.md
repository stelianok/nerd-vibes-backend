# Nerd Vibes - Backend

## Description

This is the backend of a fictional ecommerce focused on selling "geek" goods. Currently being built with NestJS and Prisma

## Tech Stack

- Typescript
- Docker
- NestJS
- Prisma
- PostgreSQL

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start
```
```bash
# watch mode
$ yarn start:dev
```
```bash
# production mode
$ yarn start:prod
```
